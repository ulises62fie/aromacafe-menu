AROMACAFÉ WEB V2

ARCHIVOS PRINCIPALES
- index.html: estructura de la página
- styles.css: colores, tamaños y diseño
- menu-data.js: productos, precios y descripciones
- app.js: buscador, filtros y menú móvil
- assets/: logo y PDF

CÓMO ACTUALIZAR PRECIOS
1. Abre menu-data.js.
2. Busca el producto.
3. Cambia únicamente el precio dentro de las comillas.
4. Guarda.

CÓMO PUBLICAR EN GITHUB/NETLIFY
Desde la terminal de VS Code, dentro de la carpeta del proyecto:

git add .
git commit -m "Actualicé la página de Aromacafé"
git push

Netlify publicará los cambios automáticamente.

IMPORTANTE
Antes de reemplazar tu página actual, guarda una copia de seguridad de la carpeta anterior.
